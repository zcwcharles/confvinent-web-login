# Getting Started

## Setup environment
1. Install [nvm](https://github.com/nvm-sh/nvm)
2. Run `nvm install 16`
3. Run `nvm use 16`
4. Install [yarn](https://classic.yarnpkg.com/lang/en/docs/install/)
5. Run `yarn`

## To start project
Run `yarn start`
